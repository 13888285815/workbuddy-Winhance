# 隐私政策

## 1. 数据收集

Winhance 是一个本地应用程序，不会收集或传输任何个人数据到外部服务器。所有操作都在本地执行，包括：

- 系统信息收集（仅用于日志记录和功能适配）
- 应用安装和卸载操作（仅在本地执行）
- 系统设置修改（仅在本地执行）

## 2. 数据存储

Winhance 仅在本地存储以下数据：

- 日志文件：存储在 `%ProgramData%\Winhance\Logs` 目录，包含应用操作记录和系统信息
- 配置文件：存储在 `%ProgramData%\Winhance` 目录，包含用户设置和偏好
- 临时文件：存储在 `%TEMP%` 目录，用于下载和安装应用

## 3. 数据保护

Winhance 采取以下措施保护用户数据：

- 日志文件定期清理（默认30天）
- 日志文件不包含个人身份信息
- 配置文件仅存储必要的设置信息
- 临时文件在操作完成后自动清理

## 4. 用户同意

使用 Winhance 即表示您同意本隐私政策。如果您不同意本政策，请不要使用 Winhance。

## 5. 第三方服务

Winhance 可能会使用以下第三方服务：

- WinGet：用于安装和管理应用
- Chocolatey：用于安装和管理应用
- GitHub API：用于获取应用的最新版本信息

这些第三方服务的使用遵循其各自的隐私政策。

## 6. 隐私政策更新

本隐私政策可能会不时更新。更新后的隐私政策将在应用中通知用户，并在 GitHub 仓库中发布。

## 7. 联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 联系我们。
